# ProvLy Knowledge

This lane is the canonical source for home inventory intake, documentation, storage, and export behavior.

## What this lane must answer

- How should a user document items for an insurance claim?
- What fields are needed for completeness?
- How do exports work?
- What does the assistant do with receipts, photos, and attachments?

## MVP truth set

- Home inventory onboarding guide.
- Item category and room taxonomy.
- Evidence and receipt guidance.
- Export formats and claim packets.
- Security posture and storage explanation.

## P0 missing items

- Inventory schema and taxonomy.
- Export format specification.
- Security and storage explanation.

## P1 missing items

- Claim-readiness checklist.
- Photo and receipt guidance.
- Backup and restore notes.

## P2 later items

- Insurer integration docs.
- Cloud storage workflow docs.
- Portability and deletion flow references.

## Retrieval hooks

- Questions about item completeness should cite this lane.
- Questions about receipts or photos should prefer operational docs, not general advice.
- Export and claim-packet questions should stay grounded in the official format spec.

## Safety notes

- Be explicit that this is not insurance advice.
- Avoid suggesting unsupported valuation logic.
- Treat privacy and storage details as sensitive.
