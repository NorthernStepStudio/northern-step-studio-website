# Neuromove Knowledge

This lane is the canonical source for routine building, caregiver workflows, and health-adjacent safety limits.

## What this lane must answer

- How do users build routines and track progress?
- What should caregivers do when things are uncertain?
- What is not medical advice?
- How should escalation or referral be handled?

## MVP truth set

- Routine builder guide.
- Caregiver workflow guide.
- Safety limits and escalation guidance.
- Feature definitions.
- Non-medical boundary docs.

## P0 missing items

- Not-medical-advice boundary.
- Safe-use guide.
- Caregiver workflow specification.

## P1 missing items

- Age-appropriateness guidance.
- Failure-mode guidance.
- Crisis handoff and resource references.

## P2 later items

- Provider handoff templates.
- Resource directory model.
- Audit/change log references.

## Retrieval hooks

- Routine questions should be answered in practical, small-step language.
- Health-adjacent questions should avoid diagnosis or treatment claims.
- If the user asks for medical certainty, the answer should refuse or redirect.

## Safety notes

- Treat this lane as health-adjacent, not diagnostic.
- Do not claim therapeutic efficacy.
- Keep child-safety and privacy considerations explicit.
