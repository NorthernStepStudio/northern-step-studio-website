# NexusBuild Knowledge

This lane is the canonical source for PC build planning, compatibility, pricing, and recommendation behavior.

## What this lane must answer

- Can these parts work together?
- What is the right build for a budget or use case?
- What happens when a part is unavailable or out of date?
- How should live pricing be interpreted?

## MVP truth set

- Build workflow guide.
- Compatibility and constraints rules.
- Budget-to-build rubric.
- Parts glossary.
- Known limitations around availability and price freshness.

## P0 missing items

- Compatibility and constraint doc.
- Budget build rubric.
- Known limitations and pricing caveats.

## P1 missing items

- Parts glossary.
- Benchmark methodology note.
- Reference build examples with revision history.

## P2 later items

- Curated comparison catalog.
- Upgrade-path examples.
- Pricing source strategy docs.

## Retrieval hooks

- Build-fit, socket, PSU, memory, and case questions should land here.
- Questions about live pricing should mention freshness and source strategy.
- Recommendation answers should always note caveats and assumptions.

## Safety notes

- Keep guidance framed as general advice.
- Avoid certainty when price or stock data is stale.
- Do not present recommendations as guarantees.
