# Agent Naming Spec

## Canonical product copy
- Matterhorn is the main NStep AI agent.
- Synox is the core intelligence engine powering Matterhorn.
- NStep AI is the broader product and ecosystem.

## UI copy (exact wording)
- Matterhorn ready
- Matterhorn is analyzing your repo
- Matterhorn proposed a patch
- Powered by Synox
- Synox runtime connected
- Synox provider: Ollama

## Usage rules
- User-facing agent labels and action copy use `Matterhorn`.
- Core engine/runtime/provider labels use `Synox`.
- Ecosystem and umbrella references use `NStep AI`.
