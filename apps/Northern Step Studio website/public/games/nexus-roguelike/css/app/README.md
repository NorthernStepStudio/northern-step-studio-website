# Stylesheet Split

`index.html` now loads:

1. `base.css`
2. `layout-controls.css`

Edit guidance:

- `base.css`: global tokens, title screen, world/map tile styles, panels, combat visuals.
- `layout-controls.css`: floating HUD controls, minimap placement, d-pad/action bar positioning.

When adjusting UI placement, change `layout-controls.css` first.
