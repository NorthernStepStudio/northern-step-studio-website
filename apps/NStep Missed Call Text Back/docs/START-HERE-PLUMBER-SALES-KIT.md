You are not selling "AI."

You are selling this:

`When a plumbing business misses a call, the system responds instantly, qualifies the lead, checks urgency, and sends the owner a clean summary so the job is not lost.`

## One-Line Pitch

`Never miss a plumbing job again.`

Use this verbatim:

`This is a 24/7 missed-call response system for plumbing businesses. When a customer calls and nobody picks up, it responds instantly, asks a few qualifying questions, figures out urgency, and sends you a clean lead summary so you do not lose the job.`

Use this verbatim:

`If you miss a call, that customer usually calls the next plumber. This fixes that by responding instantly and keeping the customer engaged until you can take over.`

## Objection Handling

### "What if it says something wrong?"

`It follows a controlled workflow and rules, so it stays on task instead of going off-script.`

### "Can I still talk to the customer myself?"

`Yes. This only handles the first response so you do not lose the lead. You still take over when you want.`

### "Is this complicated?"

`No. Once it is set up, it runs automatically in the background.`

### "What if I am already using my phone and texting customers?"

`That is fine. This handles the first missed-call recovery so you only spend time on qualified leads.`

### "How much is it?"

`The Starter setup is $500, then $200 to $300 per month depending on the setup.`

### "I need to think about it."

`Makes sense. The real question is how many missed jobs are slipping away now. If even one a week gets recovered, it usually covers itself.`
