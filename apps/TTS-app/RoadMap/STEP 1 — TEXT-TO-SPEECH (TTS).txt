🔹 Prebuilt binaries (EASIEST)

Scroll down in the repo to Releases, or go directly:
👉 https://github.com/rhasspy/piper/releases

Download based on your OS:

Windows → piper_windows_amd64.zip

macOS (Apple Silicon) → piper_macos_arm64.tar.gz

macOS (Intel) → piper_macos_x64.tar.gz

Linux → matching Linux build

💡 If you don’t want to compile anything → use releases

🔹 Voice models (VERY IMPORTANT)

Piper does NOT include voices by default.

👉 Official voice repository
https://huggingface.co/rhasspy/piper-voices

Example voices:

en_US-lessac-medium

en_US-amy-low

en_GB-alan-medium

Many languages supported

Download one .onnx + .json per voice.

🧪 QUICK TEST (after download)

Example command:

piper --model en_US-lessac-medium.onnx --output_file test.wav


Then type text and press Enter.

🧠 What “synthetic / blended voices” means (simple)

Synthetic voice → fully AI-generated (Piper, Coqui, etc.)

Blended voice → trained using multiple speakers to avoid copying a real person

Piper voices are SAFE and non-celebrity

No legal nightmare 👍
