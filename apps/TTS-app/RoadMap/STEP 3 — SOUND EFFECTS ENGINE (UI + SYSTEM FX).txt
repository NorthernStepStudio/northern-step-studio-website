3️⃣ Procedural Sounds > Static WAVs (IMPORTANT)

Procedural sounds scale better and feel modern.

Bubble Pop (Slider / Toggle)
import numpy as np

def bubble_pop(sr=44100, duration=0.08, pitch=600):
    t = np.linspace(0, duration, int(sr * duration))
    env = np.exp(-30 * t)
    freq = pitch * (1 - 0.6 * t)
    signal = np.sin(2 * np.pi * freq * t) * env
    return signal


🔥 Tiny, clean, reactive.

Click (Button)
def click_sound(sr=44100):
    noise = np.random.randn(int(sr * 0.01))
    env = np.exp(-60 * np.linspace(0, 1, noise.size))
    return noise * env

Error Buzz
def error_buzz(sr=44100):
    t = np.linspace(0, 0.2, int(sr * 0.2))
    return 0.4 * np.sin(2 * np.pi * 120 * t) * np.sign(np.sin(2*np.pi*8*t))

4️⃣ Emotion-Aware Sound Design (Huge Win)

Tie SFX to your emotion system from STEP 2.

Example:

def modulate(sound, emotion):
    sound *= 1 + 0.3 * emotion["energy"]
    sound = np.tanh(sound * (1 + emotion["anger"]))
    return sound


Now:

Calm UI → softer sounds

Energetic voice → punchy UI

Sad tone → lower pitch, longer decay

This is chef’s kiss 🧠

5️⃣ Event → Sound Mapping
SOUND_MAP = {
    "slider_move": lambda e: bubble_pop(pitch=500 + 300*e["energy"]),
    "slider_release": lambda e: bubble_pop(pitch=800),
    "button_click": lambda e: click_sound(),
    "error": lambda e: error_buzz(),
}


Trigger:

sound = SOUND_MAP[event](emotion)
play(sound)

6️⃣ Mixer (So Sounds Don’t Fight)

Simple but essential:

def mix(sounds):
    mix = np.sum(sounds, axis=0)
    return np.clip(mix, -1, 1)


Add:

Master UI volume

Ducking under TTS playback (-6 dB)

7️⃣ Latency Rules (Non-Negotiable)

UI SFX < 10 ms

Pre-generate common sounds

No disk reads on interaction

Use memory buffers

If there’s lag, the illusion dies.

8️⃣ Polish Moves (These Matter)

✔ Randomize pitch ±3%
✔ Randomize timing ±5 ms
✔ Stereo width for sliders
✔ Short reverb tail for “premium” feel

Example:

pitch *= np.random.uniform(0.97, 1.03)
